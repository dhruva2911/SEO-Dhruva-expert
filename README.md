# SereneHomes – Real Estate Landing Page

A single-page, responsive real estate landing site for gated community apartments, built with plain **HTML, CSS and JavaScript**.

It uses a **brown and cream** luxury-inspired theme, large hero imagery, sections for **Apartments**, **Gated Communities**, and **Contact Us** with a simple enquiry form.

## Structure

- `index.html` – main page with:
  - Fixed navigation bar linking to in-page sections
  - Hero section with rotating full-width apartment/community imagery and captions
  - Apartments cards featuring interiors (living room, bedroom, kitchen)
  - Gated Communities section highlighting amenities (pool, gym, parks, security)
  - Contact form with name, phone, email, interest and message fields
- `styles.css` – layout, color scheme, responsive behavior and subtle animations
- `script.js` – navigation toggle, hero image slider, scroll reveal and form feedback

## Image Requirements (4K)

For production, replace the example image URLs in `index.html` with your own **true 4K assets**:

- Recommended resolution: **3840 × 2160 px or higher**
- Categories:
  - Apartment interiors – living rooms, bedrooms, kitchens
  - Apartment exteriors / buildings
  - Gated community amenities – pools, gyms, parks, security gates

The layout is designed to showcase 4K images edge-to-edge; quality depends on the source files you provide.

## Running the Site

1. Place all files in the same folder.
2. Open `index.html` directly in your browser, or serve the folder with a simple static server (for example using VS Code Live Server or any HTTP server).
3. Test on mobile, tablet and desktop to see the responsive layout and interactions.

No external build tools or dependencies are required.
